    Installation

    1. Rename .env-sample into .env
    2. Update environment variables listed in the renamed .env to match your local setup. Don't forget to choose the NODE_ENV (development, production).
    3. npm i

    Run

    1. Mongo and DosismartCPP need to be running for the bridge to start. CF repository root README.md to start those apps.
    2. npm start

    Test

    npm test

    Documentation Generation

    Run this command to generate the jsdoc : npm run doc

    If you want to exclude a file or a folder, place its path in the exclude array of docs/conf.json like so :

        "source": {
        "exclude": [
            "node_modules",
            "test"
        ]
    },
